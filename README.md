# PO Generator

## Description
PO Generator is a Python-based application that helps users create and manage purchase orders easily. It automatically calculates totals, exports purchase orders as PDF and Excel files, and saves them based on address for quick access and better organization.

## Features
- Create professional purchase orders  
- Automatically calculate item totals and final amount  
- Export purchase orders as PDF files  
- Save purchase orders in Excel format  
- Organize and store purchase orders based on address  
- Clean and simple user interface  

## Tech Stack
- Language: Python  
- PDF Generation: ReportLab / FPDF  
- Excel Handling: Pandas / OpenPyXL  

## How It Works
- User enters purchase order, vendor, and item details  
- The application calculates totals automatically  
- Purchase order is generated and exported as a PDF  
- The same data is saved in an Excel file  
- Files are stored based on address for easy access
